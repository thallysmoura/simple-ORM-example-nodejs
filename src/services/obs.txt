
1. **create()**: Cria um novo registro no banco de dados com base nos dados fornecidos.

2. **findAll()**: Recupera todos os registros de uma tabela.

3. **findByPk()**: Recupera um registro por sua chave primária.

4. **findOne()**: Recupera um único registro que corresponde a uma condição específica.

5. **update()**: Atualiza um ou mais registros no banco de dados com base em uma condição.

6. **destroy()**: Exclui um ou mais registros do banco de dados com base em uma condição.

7. **count()**: Conta o número de registros que correspondem a uma condição específica.

8. **sum()**: Calcula a soma de um campo numérico com base em uma condição.

9. **max()**: Encontra o valor máximo de um campo numérico com base em uma condição.

10. **min()**: Encontra o valor mínimo de um campo numérico com base em uma condição.

11. **avg()**: Calcula a média de um campo numérico com base em uma condição.

12. **increment()**: Incrementa o valor de um campo numérico em uma quantidade específica.

13. **decrement()**: Decrementa o valor de um campo numérico em uma quantidade específica.

Esses são apenas alguns dos comandos nativos do Sequelize. Além desses, existem muitos outros métodos e opções disponíveis no Sequelize para trabalhar com consultas mais complexas, transações, validações, associações entre modelos e muito mais.

Para uma lista completa dos comandos nativos do Sequelize e para obter detalhes sobre cada um, você pode consultar a documentação oficial do Sequelize. Sempre que estiver trabalhando com o Sequelize, a documentação é uma valiosa fonte de referência para entender os recursos disponíveis e como utilizá-los de maneira eficiente.
